Dalplex Recreational

Dalplex Recreational is the revamped version of sports facility website of Dalhousie University. This dynamic platform seamlessly recreates all the functionalities of the old Dalplex website while integrating modern standards and a sleek UI. Built on the MEAN stack, Dalplex Recreational goes beyond by introducing new features like a payment facility for sports equipment and an efficient pre-booking management system. Experience the perfect blend of tradition and innovation, making your Dalplex experience more accessible and enjoyable than ever before.

Responsibilities:

  • Created a 3-tier application with modern UI implemented using Angular, ExpressJS, MongoDB, and NodeJS.

  • Developed RESTful API in MVC architecture using ExpressJS.

  • WApplied advanced performance evaluation and monitoring techniques to ensure an efficient web application.

  • Tech Stack: Angular, ExpressJS, , MongoDB, NodeJS, HTML, CSS, Bootstrap, NETLIFY
