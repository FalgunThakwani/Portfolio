Confidential Fetch

Confidential Fetch is a secure web app designed for centralized password storage. This user-friendly platform generates security tokens, sending them via email for authorized access. Leveraging a serverless architecture and CloudFormation scripts for automated deployment, Confidential Fetch prioritizes security. With a robust CI/CD pipeline, this app ensures a seamless and protected experience for managing your passwords confidentially.

Responsibilities:

  • Utilized AWS SNS service to send secure tokens for the application.

  • Implemented CI/CD pipeline and deployment using AWS Cloud Build.

  • Developed Lambda functions to create secure tokens for enhanced security.

  • CloudFormation scripts for automated deployment and infrastructure management.

  • Tech Stack: Angular, AWS - Lambda, EC2, CloudFormation, DynamoDB, CI/CD
